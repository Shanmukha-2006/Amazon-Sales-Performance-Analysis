# 📊 Amazon Sales Performance Analysis

## About the Project

This project analyzes Amazon sales data using Python in **Google Colab**. It explores sales trends, product performance, and customer purchasing patterns through data cleaning, analysis, and visualizations.

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Sales trend analysis
* Product category analysis
* Data visualization
* Business insights

## Technologies Used

* Google Colab
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset

The project uses an Amazon Sales dataset containing order details, product categories, sales, quantity, prices, and customer information.

## Project Workflow

1. Import the dataset into Google Colab
2. Clean and preprocess the data
3. Perform Exploratory Data Analysis (EDA)
4. Create charts and visualizations
5. Analyze sales performance
6. Generate business insights

## Results

The analysis helps identify:

* Best-selling products
* Sales trends over time
* Top-performing categories
* Customer purchasing patterns
* Key business insights
